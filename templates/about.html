03 : About me

Welcome, I'm Roman Pobotin

I lead Quality Assurance Automation with a focus on blending traditional testing with the innovative fields of SDET and TestOps. Since December 2015, I've been on a mission to transform automated testing for both web and mobile platforms, striving for seamless integration into development cycles.

What I Do:

  • Analyze products to lay down robust QA foundations.
  • Design and implement strategic test automation frameworks.
  • Develop innovative tools and solutions for complex application testing, providing analytics and solving business problems, such as comparison tools that offer actionable feedback.
  • Optimize continuous integration and insights through detailed reporting.
  • Guide and mentor my team, promoting a culture of innovation and quality. This includes not only teaching test writing but also empowering them to build their own solutions.

Technical Expertise:

My technical prowess is defined by a comprehensive skill set across various languages, tools, and systems:

  • Languages: Highly proficient in Python; foundational knowledge in JavaScript and C#
  • Tools: Expert in Selenium WebDriver, Appium, Allure; proficient in bash/PowerShell, Cypress, etc. Skilled in Playwright and Puppeteer for browser automation
  • Cloud and DevOps: Proficient in a comprehensive range of AWS services including S3, Secret Manager, Boto3, AWS Device Farm, RDS, ECS, ECR, CloudFormation, and more. Skilled in managing infrastructure as code with AWS CDK (TypeScript), and adept in using CodeCommit, CodeBuild, CodePipeline, Artifactory, and CodeArtifact. Experienced with AWS ChatBot, SNS, VPC, Security Groups, and IAM to ensure robust and secure cloud solutions.
  • Version Control: Skilled in Git
  • Continuous Integration: Experienced with Jenkins, TFS, GitHub Actions
  • Testing Frameworks: In-depth knowledge of pytest
  • Operating Systems: Versatile in Mac OS, Windows, Unix
  • Databases: Proficient in MS SQL, MySQL, and PostgreSQL
  • Computer Vision: Familiar with OpenCV for image processing and computer vision tasks in testing scenarios, such as validating canvas elements
  • Web Frameworks: Able to create simple web services and UIs with Flask; some experience with Django
  • Monitoring: Knowledgeable in using Grafana for monitoring and visualization
  • Prompt Engineering: Skilled in leveraging advanced technologies like ChatGPT, Gemini, or Amazon Q to solve complex problems and innovate beyond traditional QA tasks
